## POST-FILING CHECKLIST

### Immediate (Within 1 Week)

- [ ] Save filing receipts and serial numbers
- [ ] Set calendar reminders for status checks
- [ ] Create TSDR monitoring schedule
- [ ] Document all filing confirmations

### Ongoing (Every 2-4 Weeks)

- [ ] Check TSDR for application status
- [ ] Monitor for Office Actions
- [ ] Review any USPTO correspondence

### If Office Action Received

- [ ] Note 6-month response deadline
- [ ] Review examiner's concerns
- [ ] Prepare response (consider attorney assistance)
- [ ] File response before deadline

### Upon Publication

- [ ] Monitor Official Gazette publication
- [ ] Watch for opposition filings (30-day window)
- [ ] Prepare for registration if no opposition

### Upon Registration

- [ ] Receive registration certificate
- [ ] Begin using ® symbol
- [ ] Set maintenance reminders:
  - [ ] Section 8 Declaration (Years 5-6)
  - [ ] Section 9 Renewal (Year 10)
