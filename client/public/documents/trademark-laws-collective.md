### FILING INSTRUCTIONS

1. **Create USPTO Account:** Visit https://www.uspto.gov and create a USPTO.gov account

2. **Access TEAS:** Navigate to Trademark Electronic Application System (TEAS)

3. **Select Application Type:** Choose TEAS Plus for lower fees (if goods/services match ID Manual)

4. **Complete Application:**
   - Enter mark information exactly as shown above
   - Select all applicable classes
   - Enter goods/services descriptions
   - Upload specimens for each class
   - Pay filing fees

5. **Monitor Application:** Check status regularly at TSDR (Trademark Status & Document Retrieval)

6. **Respond to Office Actions:** If USPTO issues an Office Action, respond within 6 months
