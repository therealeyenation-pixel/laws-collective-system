## SOFTWARE LICENSE MANAGEMENT - PROPERTY MANAGEMENT RESPONSIBILITY

### Centralized License Management (Property Management Department)

The Property Management department is responsible for centralized software license management across all departments. This includes:

**Asset Inventory & Tracking:**
- Maintain master inventory of all software licenses
- Track license assignments to users and departments
- Monitor license utilization and compliance
- Manage license transfers between employees

**Renewal Management:**
- Track expiration dates for all subscriptions
- Process renewals in advance of expiration
- Negotiate enterprise pricing and volume discounts
- Evaluate renewal vs. replacement decisions

**Cost Allocation:**
- Allocate software costs to appropriate departments
- Track software spending against department budgets
- Generate cost reports for financial planning
- Identify cost optimization opportunities

**Compliance Monitoring:**
- Ensure license compliance across all platforms
- Conduct quarterly software audits
- Manage vendor relationships and contracts
- Maintain documentation for audits

**Onboarding/Offboarding:**
- Provision software access for new employees
- Revoke access upon employee separation
- Transfer licenses as needed
- Maintain access logs

## SOFTWARE MANAGEMENT POLICIES

### License Management
1. All software licenses are registered under @lawscollective.org domain
2. CEO/Founder maintains master admin access to all platforms
3. Department managers have admin access within their department
4. Property Management tracks all licenses centrally
5. Licenses can be reassigned as needed

### Portability
1. Cloud-based software can be accessed from any authorized device
2. Users log in with organizational credentials
3. Perpetual licenses (DaVinci, Final Cut, etc.) are tied to organization
4. Software can be installed on multiple devices per license terms

### Budget Authority
1. Department managers have authority to manage software within allocated budget
2. New software requests over $500/year require CEO approval
3. Quarterly software audit conducted by Property Management
4. Annual budget review and adjustment

### Security
1. All accounts use organizational SSO where available
2. Two-factor authentication required on all platforms
3. Regular password rotation (90 days)
4. Immediate access revocation upon separation
